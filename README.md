# サイゼリヤシフト管理システムの試作


## 作成の目的
サイゼリヤの毎月のシフト申請が、全て紙媒体で行われている。

このことについて、かなりの不便さを感じたため、自身の経験値にもなりうるため、システム開発を私の店長に提案し、着手を決意しました。

- 体調不良やインターンシップの参加等で、長期間シフトが入らず、申請期限までに申請できない人がいること。また、今いなくてもいずれ起こりうる課題であること。

-記入ミスがあった際に二重線で消すなどして、結果としてわかりづらくなってしまうことが多々あること。

-また、紙でのシフト申請内容や、曜日ごとの勤務時間の変更内容(学生は頻繁に時間が変わりやすい点)を、店長が手作業でデータを入力している事。

-これらにより、シフトが組まれた際に、組間違いが起こりやすいと感じた。
